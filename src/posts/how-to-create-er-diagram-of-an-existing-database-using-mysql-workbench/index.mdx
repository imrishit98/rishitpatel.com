<p>
  The following step-by-step guide will help you&nbsp;
  <strong>create ER Diagram of an existing database</strong>&nbsp;using&nbsp;
  <a href="https://www.mysql.com/" target="_blank" rel="noreferrer noopener">
    MySQL
  </a>
  &nbsp;Workbench very easily.
</p>

<h3 id="5d98">Step 1:</h3>

<p>
  <strong>
    Open your MySQL Workbench and click on Database. (You can also press Ctrl+R
    and skip first two steps)
  </strong>
</p>

<h3 id="04a1">Step 7:</h3>

<p>
  <strong>
    Now there will be option to select MySQL table objects to reverse engineer,
    it’s the objects for which you want to create the diagram, After selecting
    objects click on Execute.
  </strong>{" "}
  <strong>
    (Make sure that checkbox at bottom-left corner labeled as “Place imported
    objects on a diagram is checked)
  </strong>
</p>
